### To run
+ Reacted Project created using Vite
+ cd into directory
+ run <br> 
    ``` bash 
        > npm install
    ```
+ Once all the dependencies are installed. <br>
     ``` bash 
        > npm run dev //run localhost
    ```
+ To deploy on Production 
     ``` bash 
        > npm build release 
        //Export & upload /dist folder on web hosting.
    ```
+ That's it.
